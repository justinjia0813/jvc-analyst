## 正文级亮点

### 1. 行业趋势

| 字段 | 内容 |
| --- | --- |
| 标题级亮点 | 中小制造企业质检自动化可能进入“愿意尝试低门槛方案”的窗口。 |
| 事实依据 | deck 将目标客户限定在 3C 零部件、注塑件、金属件工厂，并反复强调人工漏检、招工难和交付周期痛点。[示例素材: deck p.3-p.4] |
| 判断论点 | 如果这些客户过去被传统视觉方案的成本和交付复杂度挡在门外，那么低门槛方案可能打开一个此前没有被充分服务的细分市场。[推测] |
| 仍需验证 | 客户是否真的有预算，且替换人工或传统方案后能形成可量化 ROI。 |

### 2. 技术节点

| 字段 | 内容 |
| --- | --- |
| 标题级亮点 | 低代码训练 + 边缘部署如果成立，可能降低传统视觉项目的交付门槛。 |
| 事实依据 | 产品页展示“样本上传、标注、训练、边缘部署”的流程，并称工艺工程师可参与训练。[示例素材: deck p.5-p.6] |
| 判断论点 | 技术亮点不在算法概念本身，而在能否把算法交付从专家工程变成半标准化流程。[推测] |
| 仍需验证 | 样本量、误检/漏检率、部署人天、跨产线复用率。 |

### 3. 团队优势

| 字段 | 内容 |
| --- | --- |
| 标题级亮点 | 团队来自视觉交付一线，可能比纯算法团队更理解工厂场景。 |
| 事实依据 | CEO 曾在机器视觉公司负责客户方案，CTO 有制造业算法交付经验。[示例素材: deck p.10] |
| 判断论点 | 工业 AI 的早期关键不是模型演示，而是场景拆解、客户沟通和交付收敛；相关交付经历可能减少试错成本。[推测] |
| 仍需验证 | 过往项目清单、客户评价、团队是否具备规模化销售组织能力。 |

### 4. 商业化进展

| 字段 | 内容 |
| --- | --- |
| 标题级亮点 | 已有产线部署和年化收入线索，说明不是纯概念阶段。 |
| 事实依据 | 创始人称已覆盖 18 条产线、年化收入 420 万元。[创始人自述][未核实] |
| 判断论点 | 即使数字仍需查证，早期付费和产线部署线索可以作为商业化验证入口，而不是只看产品 demo。[推测] |
| 仍需验证 | 合同、回款、收入拆分、客户续费、同客户扩产线记录。 |
